# Feasibility Simplex Solver

Implementacija algoritma je urađena u **C++**. Algoritam određuje zadovoljivost sistema linearnih ograničenja u nenegativnom skupu rešenja svodeći problem na minimizaciju.

Projekat je razvijen kao **seminarski rad iz predmeta „Automatsko rezonovanje“** na master studijama **Matematičkog fakulteta, Univerziteta u Beogradu**.

---

## Izgradnja

```bash
cmake -S . -B <build_dir>
cmake --build <build_dir>
```
---

## Pokretanje

```bash
./<build_dir>/feasibility_simplex_solver <input_file_path>.fs
```
